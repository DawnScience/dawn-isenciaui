Export

A set of Wiki documents can be exported as HTML to a external directory. All the linked Eclipse resources will also be exported with the HTML to form a self-contained snapshot of the Wiki. In addition, Java source will be converted to HTML using Java2Html.

To do an export:

  1. select Export from the File menu, or right clicking the folder containing the Wiki you want to export and select Export
  2. select Wiki in the Export dialog and click Next
  3. select the Wiki folder you want to export and set the destination folder
  4. click Finish
