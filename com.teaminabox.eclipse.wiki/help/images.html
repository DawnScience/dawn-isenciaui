Images

The browser view and exported HTML documents will inline links to images. For example:

http://www.eclipse.org/images/EclipseBannerPic.jpg

will embed the image in place in the document.

Local Images

If you would like a set of local images made available to your documents then the you can use the Eclipse: protocol to link to them and they will be inlined. But you may find it useful to create an entry in your Wikispace to the image location:

Images=Eclipse:/MyProject/MyDocs/images/

You can now link to images with: Images:myImage.jpg
